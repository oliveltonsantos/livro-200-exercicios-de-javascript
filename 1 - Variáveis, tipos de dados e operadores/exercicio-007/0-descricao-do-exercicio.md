## Descrição do exercício:


Declare três variáveis e atribua valores numéricos a elas. Use operadores de comparação para comparar os valores entre as variáveis. Imprima os resultados.


## Desafio: Quem é maior, menor ou igual?

### Contexto

Você vai comparar três números digitados pelo usuário e descobrir:
- qual é o maior
- qual é o menor
- se existem valores iguais

Usando operadores de comparação (>, <, >=, <=, ===, !==).


### Entrada do usuário

Crie três inputs:
    - Número A
    - Número B
    - Número C

E um botão:
    - Comparar valores


### Regras do exercício

Quando o usuário clicar no botão:

1 - Verifique se todos os campos foram preenchidos

2 - Converta os valores para Number

3 - Compare os valores usando operadores de comparação

4 - Mostre no HTML frases como:
    - “A é maior que B”
    - “B é igual a C”
    - “C é o menor valor”


### O que esse exercício treina (sem você perceber):

 - Conversão de tipos (Number)
 - Comparações reais (não decoradas)
 - Fluxo lógico
 - Clareza de leitura do código
