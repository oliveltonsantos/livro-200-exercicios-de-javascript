## Descrição do exercício:

Escreva um programa que calcula o IMC (Índice de Massa Corporal) de uma pessoa e imprime uma mensagem indicando se a pessoa está abaixo do peso, com peso normal, com sobrepeso ou obesa.

Nesse exercício, você vai criar uma função que recebe o peso (em kg) e a altura (em m) de uma pessoa, calcula o IMC e verifica em qual faixa o valor se encaixa. 

As faixas são: 

    - Abaixo do peso (IMC < 18.5)
    - Normal (18.5 ≤ IMC < 25)
    - Sobrepeso (25 ≤ IMC < 30)
    - Obesidade (IMC ≥ 30)
